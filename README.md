# storybook-vvd
storybook + vite + vue + docker

## Clone & Install Repository
```
git clone https://github.com/difuoan/storybook-vvd
cd storybook-vvd/
make init
make up
```

## Run Repository
```
make up
```

## All `make` Commands
```
init # Add default .env
up
start
down
stop
tail # get logs
build
restart
clear # Remove .env
ps # Lists containers for a Compose project, with current status and exposed ports.
```
